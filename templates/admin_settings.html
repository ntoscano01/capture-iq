Security Settings
attempts
Number of consecutive failed login attempts before an account is automatically locked. A locked account must be manually unlocked by an admin from the Users page.
characters
Minimum number of characters required when creating or resetting a password. Minimum allowed value is 4.
Setting Reference

Lockout Threshold: Industry standard is 5 attempts. Lower values (3) are stricter and better for sensitive data. Higher values (10) reduce accidental lockouts for users who mistype frequently.

Password Length: 8+ characters is recommended for most deployments. The current minimum of {{ settings.min_password_length }} characters applies to new accounts and password resets — existing passwords are not affected until reset.
