SBIR.gov — Topics REST API

Pull open solicitation topics from SBIR.gov, including full descriptions, objectives, and phase details. Covers NASA, DOD, NIH, DOE, and all other agencies.

DoD SBIR/STTR — dodsbirsttr.mil JSON API

Pulls topics directly from the DoD SBIR/STTR Innovation Portal (DSIP) REST API. Enter the BAA identifier from the portal URL (e.g. DOD_SBIR_2026_P1_CBZ).

Portal URL pattern:
https://www.dodsbirsttr.mil/topics-app/?baa=BAA_ID
Copy from the ?baa= parameter in the portal URL.
Navy SBIR — navysbir.com Web Scraper

Scrapes full BAA topic details from navysbir.com, including Objective, Description, Phase I/II/III, Keywords, Tech Areas, and CMMC level.

Update this URL when a new BAA is released (e.g. topics26_2.htm).
